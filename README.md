# Blog Post Platform

A full-stack blog post website with a modern frontend and a RESTful backend. The application supports user authentication, post management, and comments using a clean API design.

---

## Tech Stack

### Frontend
- **TypeScript**
- **React**
- **Vite**
- **Tailwind CSS**

### Backend
- **Node.js**
- **Express**
- **TypeScript**
- **JWT Authentication**
- 
---

## Features

- User authentication (register / login)
- JWT-based protected routes
- Create, read, update, and delete blog posts
- Comment system
- Like / Dislike posts
- RESTful API design
- Typed codebase (TypeScript on both ends)
